## 🎨 Phase 2: Design System & UI/UX (The "Eye-Catchy" Factor)
*Lentera must look like a AAA indie game, not a generic crypto app.*

1.  **Visual Identity:**
    *   **Theme:** "Neon-Ethnic" (Modern cyberpunk mixed with Indonesian traditional motifs like Batik patterns in the UI borders).
    *   **Typography:** Keep `Bricolage Grotesque` for displays, but add `Pixel Code` for "terminal-style" game feedback to enhance the dev vibe.
    *   **Color Palette:** Use the existing `cyan`, `pink`, and `purple` but add **"Depth Layers"** using semi-transparent glassmorphism and glow effects.
2.  **Design System Overhaul:**
    *   **Object Shapes:** Avoid standard rectangles. Use **Organic/Squircle** shapes for cards and **Asymmetric layouts** for the dashboard.
    *   **Characters:** Replace emojis with high-quality SVG or PNG sprites for Komodo, Wayang Owl, etc.
    *   **Animations:** Use `Framer Motion` for page transitions. Every button must have a "Haptic-like" visual feedback (slight scale down on click, glow on hover).
